Filtering

Use url parameter ?model= to filter on model.

For example: https://metabento.com/nodes?model=ICDC

JSON

Use url parameter ?format=json to get JSON-formatted output.

For example: https://metabento.com/valuesets/EapU3g?format=json

ID

Individual entities (nodes, properties, and terms) are identified by their Id either in the URL path or as parameter (as a query string)
  • In the URL path: /terms/<termid>
  • As a query parameter: /terms?id=termid

For example:

  • https://metabento.com/valuesets/EapU3g
  • https://metabento.com/valuesets?id=EapU3g
